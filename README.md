# Dockerizing-django

Dockerizing Django

INCLUDED:

- Docker
- NGINX
- POSTGRES
- [POETRY](https://python-poetry.org/docs/)
- REDIS
- CELERY

clone the repo and run:

```
    docker-compose up -d --build
    docker-compose up
```
